The TYPO3 community in Freiburg / Germany organizes a code sprint from Friday, July 13th - Sunday, July 15th.

Topics

The main topic planned are:

  • New Logging API for the TYPO3 core (36 chambers!)
  • FLOW3 PackageBuilder
  • TYPO3 ExtensionBuilder
  • PHP Parser and backporter package for FLOW3/TYPO3

Of course any other TYPO3 / FLOW3 / Phoenix related topics are possible.

Participate

As of now seven developers from Switzerland and Germany have already confirmed to participate. If you like to spend the weekend with TYPO3 / FLOW3 in the sunniest corner of Germany then get on board! Remote reviews are also very welcome.

If you want to join, put yourself onto the Codesprint page in the TYPO3 wiki and get in touch with Steffen Müller or Nico de Haen.

Sponsoring

The event will be sponsored by /gebrüderheitz and ndh websolutions and jweiland.net. We are still looking for one or more sponsors to spend the travel costs for up to 4 participants.

If you want to become a sponsor, get in touch with Steffen Müller or Nico de Haen.
